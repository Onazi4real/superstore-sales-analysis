# Superstore Sales Analysis Project

## Project Overview
This project analyzes the Superstore sales dataset using Python to uncover business insights from sales, profit, customers, products, regions, and time trends.

## Objectives
- Analyze sales and profit performance
- Identify top-performing products and customers
- Analyze categories, regions, and states
- Explore yearly, quarterly, and monthly trends
- Generate professional charts for reporting
- Prepare the project for an interactive dashboard

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- OpenPyXL
- Microsoft Word
